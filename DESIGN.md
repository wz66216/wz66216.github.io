# 网站设计规范文档

## 一、项目概览

| 项目 | 说明 |
|------|------|
| **站长身份** | 物理学生 + 准专业级国际象棋棋手 |
| **网站定位** | 个人博客 / 作品集（混合型） |
| **核心内容** | 物理笔记、国际象棋文章、生活随笔 |
| **语言** | 中英双语，用户可切换 |
| **设计风格** | 瑞士网格 (Swiss Grid) + 国际象棋深度融合 |

---

## 二、视觉风格

### 2.1 设计基因

**瑞士网格设计** × **国际象棋美学**

- 严格的 12 列网格系统，模块化排版
- 无衬线粗体标题 + 细体正文
- 黑白为绝对主色，棋盘纹理贯穿全站
- 几何线条分割，无多余装饰
- 统一的垂直节奏与间距系统

### 2.2 配色系统

#### Light Mode（白方）

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景（白格） | `#FFFFFF` | 主背景 |
| 背景（黑格） | `#F0F0F0` | 交替区块背景 |
| 主文字 | `#1A1A1A` | 接近纯黑 |
| 次要文字 | `#6B6B6B` | 灰色辅助文字 |
| 边框/分割线 | `#E0E0E0` | 细线分割 |
| 强调色 | `#1A1A1A` | 纯黑加粗，无彩色强调 |

#### Dark Mode（黑方）

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景（黑格） | `#121212` | 主背景 |
| 背景（白格） | `#1E1E1E` | 交替区块背景 |
| 主文字 | `#F5F5F5` | 接近纯白 |
| 次要文字 | `#9E9E9E` | 灰色辅助文字 |
| 边框/分割线 | `#333333` | 细线分割 |
| 强调色 | `#FFFFFF` | 纯白加粗 |

### 2.3 字体

| 用途 | 字体 | 字重 |
|------|------|------|
| 英文标题 | Inter / Helvetica Neue | 700-900 (Bold/Black) |
| 中文标题 | 思源黑体 (Noto Sans SC) | 700 (Bold) |
| 英文正文 | Inter | 400 (Regular) |
| 中文正文 | 思源黑体 (Noto Sans SC) | 400 (Regular) |
| 代码/LaTeX | JetBrains Mono | 400 |

### 2.4 间距系统

基础单位：`8px`

| 级别 | 值 | 用途 |
|------|-----|------|
| xs | 4px | 行内元素间距 |
| sm | 8px | 紧凑元素间距 |
| md | 16px | 段落间距 |
| lg | 32px | 板块内间距 |
| xl | 64px | 板块间间距 |
| 2xl | 128px | 大区块分隔 |

## 三、国际象棋元素融合方案

### 3.1 棋盘纹理

- **全站贯穿**：黑白交替的区块背景模拟棋盘格，每个内容板块交替使用"白格"和"黑格"背景色
- **Hero 区**：全屏棋盘纹理背景，8×8 网格以微妙透明度铺底
- **博客卡片**：黑白交替排列，模拟棋盘视觉
- **分割线**：使用棋盘格纹样式替代普通分割线

### 3.2 棋子等级制（技能展示）

用国际象棋棋子表示技能熟练度：

| 棋子 | 等级 | 含义 |
|------|------|------|
| ♟ 兵 (Pawn) | 入门 | 刚接触，基础了解 |
| ♞ 马 (Knight) | 初级 | 能够使用，有一定经验 |
| ♝ 象 (Bishop) | 中级 | 较为熟练，独立使用 |
| ♜ 车 (Rook) | 高级 | 深入理解，丰富经验 |
| ♛ 后 (Queen) | 精通 | 专家级，核心技能 |
| ♚ 王 (King) | 核心 | 定义身份的核心能力 |

### 3.3 动效设计

| 场景 | 动效 | 说明 |
|------|------|------|
| 页面加载 | 棋子入场 | 元素像棋子落子一样从上方"放置"到位 |
| 滚动触发 | 棋子移动轨迹 | 内容块沿 L 形（马步）或直线（车步）滑入 |
| 卡片 Hover | 棋盘格翻转 | 卡片微微翻转，背景黑白反转 |
| 主题切换 | 棋盘翻转 | 整体画面以棋盘翻转动画过渡黑白方 |
| 导航当前项 | 棋子标记 | 当前页面在侧边栏用棋子图标标记位置 |

## 四、页面结构

### 4.1 整体布局

```
┌──────┬─────────────────────────────────────┐
│      │                                     │
│ 侧   │           主内容区域                 │
│ 边   │                                     │
│ 栏   │   (首页为单页滚动，子页为独立页面)     │
│      │                                     │
│ 固   │                                     │
│ 定   │                                     │
│      │                                     │
└──────┴─────────────────────────────────────┘
```

**侧边栏宽度**：`240px`（桌面端固定），移动端折叠为汉堡菜单

### 4.2 左侧侧边栏

```
┌────────────┐
│            │
│  Logo/头像  │  ← 个人 Logo 或姓名缩写
│            │
├────────────┤
│            │
│  首页       │  ← 导航链接
│  博客       │     当前页面用棋子标记 ♚
│  关于       │
│            │
├────────────┤
│            │
│  中/EN     │  ← 语言切换
│            │
└────────────┘
```

- 背景色略深于主内容区，形成层次
- 导航链接使用无衬线粗体
- 当前页面项左侧显示棋子图标标记
- 底部放置语言切换按钮

### 4.3 首页（单页滚动）

从上到下共 4 个板块，按以下顺序排列：

#### 板块 1：Hero 开屏区（棋盘全屏型）

```
┌─────────────────────────────────────┐
│ ┌──┬──┬──┬──┬──┬──┬──┬──┐          │
│ ├──┼──┼──┼──┼──┼──┼──┼──┤          │
│ │     （8×8 棋盘纹理背景）    │          │
│ ├──┼──┼──┼──┼──┼──┼──┼──┤          │
│ │                        │          │
│ │    [姓名 / 大标题]      │          │
│ │    物理学生 · 棋手       │          │
│ │                        │          │
│ └──┴──┴──┴──┴──┴──┴──┴──┘          │
│                                     │
│         ↓ 向下滚动                   │
└─────────────────────────────────────┘
```

- 全屏高度 (100vh)
- 8×8 棋盘纹理以 `opacity: 0.06` 铺满背景
- 居中显示姓名（超大字号，900 字重）
- 副标题：一句话身份描述
- 底部有向下滚动引导箭头

#### 板块 2：关于我 (About)

```
┌─────────────────────────────────────┐
│                                     │
│  关于我 / ABOUT                     │  ← 板块标题（粗体，左对齐）
│  ─────────────────                  │  ← 棋盘格分割线
│                                     │
│  简洁的 2-3 段自我介绍               │
│  物理学习背景 + 国际象棋经历          │
│                                     │
└─────────────────────────────────────┘
```

- 背景色：白格（Light）/ 黑格（Dark）
- 文字最大宽度 `720px`，保证阅读舒适度
- 简洁段落式，不做复杂布局

#### 板块 3：技能 (Skills) — 棋子等级制

```
┌─────────────────────────────────────┐
│                                     │
│  技能 / SKILLS                      │
│  ─────────────────                  │
│                                     │
│  ♚ 王  国际象棋          ♛ 后  物理   │
│  ♜ 车  Python           ♝ 象  LaTeX  │
│  ♞ 马  Web开发           ♟ 兵  ...    │
│                                     │
│  （网格排列，每格含棋子+技能名+描述） │
│                                     │
└─────────────────────────────────────┘
```

- 交替背景色区块（黑格背景）
- 网格布局：桌面端 3 列，平板 2 列，手机 1 列
- 每个技能卡片：棋子图标 + 技能名称 + 简短描述
- 棋子从高到低排列，视觉上形成等级层次

#### 板块 4：最新文章 (Blog) — 棋盘式卡片

```
┌─────────────────────────────────────┐
│                                     │
│  最新文章 / RECENT POSTS            │
│  ─────────────────                  │
│                                     │
│  ┌──────────┐ ┌──────────┐          │
│  │ 白格卡片  │ │ 黑格卡片  │          │
│  │ 文章标题  │ │ 文章标题  │          │
│  │ 日期·分类 │ │ 日期·分类 │          │
│  └──────────┘ └──────────┘          │
│  ┌──────────┐ ┌──────────┐          │
│  │ 黑格卡片  │ │ 白格卡片  │          │
│  │ 文章标题  │ │ 文章标题  │          │
│  │ 日期·分类 │ │ 日期·分类 │          │
│  └──────────┘ └──────────┘          │
│                                     │
│        [查看全部文章 →]              │
│                                     │
└─────────────────────────────────────┘
```

- 2 列网格，黑白交替排列（如棋盘）
- 每张卡片包含：文章标题、日期、分类标签
- 分类标签用不同棋子图标区分：
  - ♜ 物理笔记
  - ♞ 国际象棋
  - ♝ 随笔/生活
- Hover 时卡片背景黑白反转
- 底部"查看全部文章"链接

#### 页脚 (Footer)

```
┌─────────────────────────────────────┐
│  © 2026 [姓名]                      │
└─────────────────────────────────────┘
```

- 极简一行式
- 仅包含版权信息
- 棋盘格分割线作为上边界

## 五、响应式断点

| 断点 | 宽度 | 布局变化 |
|------|------|----------|
| 桌面 | ≥1024px | 侧边栏固定 240px + 主内容区 |
| 平板 | 768-1023px | 侧边栏折叠为汉堡菜单，主内容区全宽 |
| 手机 | <768px | 汉堡菜单，卡片单列，间距缩小 |

### 移动端适配要点

- 侧边栏折叠为顶部汉堡菜单，点击展开全屏覆盖导航
- 博客卡片从 2 列变为 1 列
- 技能网格从 3 列变为 1 列
- Hero 区保持全屏但字号缩小
- LaTeX 公式支持水平滚动避免溢出

## 六、特殊功能需求

### 6.1 LaTeX 公式渲染

- 重度使用，需支持行内公式 `$...$` 和块级公式 `$$...$$`
- 推荐使用 KaTeX（性能优于 MathJax）
- 公式字体与正文协调

### 6.2 中英双语

- 侧边栏底部设语言切换按钮（中/EN）
- 所有板块标题为双语显示（如"关于我 / ABOUT"）
- 文章内容按语言版本独立维护

### 6.3 主题切换（黑方/白方）

- 支持亮色（白方）和暗色（黑方）两套主题
- 切换时播放棋盘翻转过渡动画
- 用户选择持久化存储到 localStorage
- 侧边栏或 Hero 区放置切换入口

## 七、页面地图

```
首页 (/)
├── #hero          → Hero 开屏区（棋盘全屏）
├── #about         → 关于我
├── #skills        → 技能（棋子等级制）
└── #blog          → 最新文章（棋盘式卡片）

博客 (/blog)
├── 列表页         → 分类筛选 + 棋盘式卡片
└── 详情页 (/blog/:slug) → 文章正文 + LaTeX

关于 (/about)      → 简洁段落 + 联系方式
```

## 八、设计关键词总结

> **严谨 · 黑白 · 棋盘 · 网格 · 几何 · 无衬线 · 粗体 · 克制 · 对比**

整站气质：像一盘精心布局的国际象棋——每个元素都有其位置，每个空间都有其意义。
