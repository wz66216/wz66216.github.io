我的第一篇博客：从零开始学建站

2026年2月14日 学习笔记

今天，我踏上了学习网站开发的旅程。作为一个完全的零基础新手，我想把整个学习过程记录下来，希望能帮助到同样想入门的朋友。

为什么要学建站？

在这个数字时代，拥有一个自己的网站就像拥有一个属于自己的小天地。你可以在上面分享知识、展示作品、记录生活，不受任何平台的限制。

"拥有自己的网站，就像在互联网上拥有一块属于自己的土地。"

网站的三大核心技术

搭建网站需要了解三门技术，它们各司其职：

1. HTML - 网页的骨架

HTML（HyperText Markup Language，超文本标记语言）定义了网页的结构和内容。所有的文字、图片、链接都是通过 HTML 标签来组织的。

<!-- 这是一个简单的 HTML 示例 -->
<h1>这是标题</h1>
<p>这是一个段落。</p>
<a href="https://example.com">这是一个链接</a>

2. CSS - 网页的外观

CSS（Cascading Style Sheets，层叠样式表）负责网页的视觉呈现。颜色、字体、布局、动画效果都由 CSS 控制。

/* CSS 让标题变成蓝色，字号 24px */
h1 {
  color: blue;
  font-size: 24px;
}

3. JavaScript - 网页的行为

JavaScript 让网页具有交互能力。按钮点击、表单验证、动态内容加载都靠它。

// 点击按钮时弹出提示
document.querySelector('button').addEventListener('click', function() {
  alert('你点击了按钮！');
});

有用的学习资源

  • MDN Web Docs（developer.mozilla.org）— 最权威的 Web 技术文档
  • W3Schools（w3schools.com）— 适合入门的交互式教程
  • 菜鸟教程（runoob.com）— 中文前端学习网站
  • FreeCodeCamp（freecodecamp.org）— 免费的编程学习平台
